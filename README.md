# tk-calc
A graphical calculator written in Python/Tk.

## Prerequisites

Tkinter may not come bundled with your installation of Python. If not, you can install it with your system's package manager, e.g.:

```
apt install python3-tk
```

## Usage
Open the main file with python:

```
python3 main.py
```

or execute the bash script:

```
./run.sh
```
